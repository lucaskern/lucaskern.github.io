Drop Dodger

Description:

A game made with the Processing Java library. The player is caught in a downpour at night and must keep the match lit by avoiding falling water

Technologies used:

Processing, Audacity, Illustrator, Photoshop

The Vision

A fast paced procedural game that is fun to play first and foremost

Ambience and good sound/visual design were initial aims of the project

The Method

Droplets are randomly spawned in and have varying speeds. Use of 2d array for drop location.

The "Play gamestate" is dynamic, in that it takes in different values depending on the difficulty the user has chosen.
